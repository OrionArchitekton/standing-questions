# Standing Questions

Ask once, get a living chart. A chat agent where the answer is a visual,
self-updating artifact: the agent keeps watching your question against live
data in ClickHouse and opens the conversation with a rendered delta when
reality changes. Built for the ClickHouse & Trigger.dev Virtual Summer
Hackathon 2026 ("Beyond the Wall of Text").

All code in this repository was written during the hackathon build window
(July 17-23, 2026). Built with Claude Code as the coding assistant
(permitted and encouraged by the event rules).
